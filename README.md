SuperHeros
Desarrollado con ReactJS

DESCRIPCIÓN: Es un proyecto en base a un ejercicio de Alkemy.

Para validar la request del login, email:challenge@alkemy.org, password: react

Funcionalidades del proyecto: El proyecto se trata de armar dos equipos de superhéroes, 3 con alineación buena y 3 con alineación mala, los superhéroes neutrales no se pueden añadir a los equipos, no se puede añadir dos veces el mismo superheroe al equipo y tampoco se pueden añadir más de tres héroes a un equipo, una vez armados los equipos se debe poder ingresar al detalle de cada superhéroe agregado al equipo para mostrar más información de el personaje elegido. Para el pryecto se utilizó la API https://superheroapi.com/.
